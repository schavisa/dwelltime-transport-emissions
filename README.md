# dwelltime-transport-emissions


## 🎯 Project Overview
This project provides a scalable, cloud-native, and fieldwork-free geomatics workflow designed to calculate **Scope 3 transport emissions** and **regional community congestion metrics** for industrial assets. 

By utilizing open-source spatial boundaries (asset polygons) and vehicle telemetry streams, this pipeline replaces expensive on-the-ground auditing with automated, programmatic location intelligence. This approach directly bridges the gap between raw geomatics engineering and corporate sustainability (ESG) compliance.

---

## 📈 Objectives & Goals
* **Automated Spatial Attribution:** Programmatically extract industrial asset perimeters using public web registries without manual field mapping.
* **Telemetry Analytics:** Isolate heavy-duty truck dwell times and bottleneck patterns within asset boundaries using time-series point-in-polygon calculations.
* **ESG Metric Translation:** Convert raw spatial temporal metrics into verifiable corporate carbon liabilities (Scope 3) and localized social impact indicators (community traffic stress).
